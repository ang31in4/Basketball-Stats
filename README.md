# About
This program uses data from [nba_api](https://github.com/swar/nba_api) to visualize NBA and WNBA players' statistics.

It generates a various bar and line graphs that compare players' points, rebounds, assists, steals, and blocks. Stats are graphed using Plotly, and saved as an HTML file with an according name.

## Instructions

1. Install all required packages:
<pre>pip install -r requirements.txt</pre>

1. In folder Basketball-Stats/code, run:
<pre>python main.py</pre>

3. Then, follow the instructions in the terminal.
